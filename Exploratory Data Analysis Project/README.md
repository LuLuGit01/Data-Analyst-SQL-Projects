# Exploratory Data Analysis (SQL Project)



## 📌 Project Overview



This project focuses on analyzing sales data using SQL. The dataset follows a star schema structure with fact and dimension tables, allowing business performance analysis across customers, products, and geography.



The goal of this project is to:

- Analyze overall business performance

- Generate key sales KPIs

- Perform customer and product analysis

- Identify top and worst performing products



- Analyze geographic sales distribution

- Build aggregated business reports using SQL



## 🗂️ Database Structure



The project uses the following tables:



**fact_sales**

- order_number – Unique order ID

- order_date – Date of order

- product_key – Product identifier

- customer_key – Customer identifier

- quantity – Items sold

- price – Selling price

- sales_amount – Total sale value



**dim_customers**

- customer_key – Customer ID

- first_name – Customer first name

- last_name – Customer last name

- birthdate – Customer birth date

- gender – Customer gender

- country – Customer country



**dim_products**

- product_key – Product ID

- product_name – Product name

- category – Product category

- cost – Product cost

## 🧾 Sample SQL Queries



🔹 1. First and Last Order Date


```sql
SELECT 

    MIN(order_date) AS first_order_date,

    MAX(order_date) AS last_order_date,

    TIMESTAMPDIFF(year, MIN(order_date), MAX(order_date)) AS order_range_years

FROM fact_sales;
```


🔹 2. Total Sales


```sql
SELECT 

    SUM(sales_amount) AS total_sales 

FROM fact_sales;
```


🔹 3. Total Orders


```sql
SELECT 

    COUNT(DISTINCT order_number) AS total_orders

FROM fact_sales;
```


🔹 4. Revenue by Category


```sql
SELECT 

    p.category,

    SUM(f.sales_amount) total_revenue

FROM fact_sales AS f

LEFT JOIN dim_products AS p

ON p.product_key = f.product_key

GROUP BY p.category

ORDER BY total_revenue DESC;
```


🔹 5. Top 5 Products by Revenue


```sql
SELECT 

    p.product_name,

    SUM(f.sales_amount) AS total_revenue

FROM fact_sales AS f

LEFT JOIN dim_products AS p

ON p.product_key = f.product_key

GROUP BY p.product_name

ORDER BY total_revenue DESC

LIMIT 5;
```


🔹 6. Top Customers by Orders


```sql
SELECT

    c.customer_key,

    c.first_name,

    c.last_name,

    COUNT(DISTINCT order_number) AS total_orders

FROM fact_sales AS f

LEFT JOIN dim_customers AS c

ON c.customer_key = f.customer_key

GROUP BY 

    c.customer_key,

    c.first_name,

    c.last_name

ORDER BY total_orders DESC

LIMIT 10;
```
